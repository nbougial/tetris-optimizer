# Task Test Files

This folder contains test-spec files mapped 1:1 with the task files in `documentation/tasks/`.

## Naming
- Task file: `documentation/tasks/NN_<name>.md`
- Test file: `documentation/tests/NN_<name>_test.md`

## How to use
1. Open the matching task and test file.
2. Execute the listed checks.
3. Mark pass/fail and attach evidence (command output or commit hash).
